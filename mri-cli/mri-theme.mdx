# mri theme :theme

## Basic usage

```bash

// 创建一个叫做demo的主题
$ mri theme demo

// 创建主题demo并启动开发模式
$ mri theme demo --start

// 删除demo主题
$ mri theme demo --remove

```

```
Usage: mri-theme <name> [options]

  创建管理 mri theme

  Options:

    --start     创建好同时启动该theme
    --remove    删除该theme
    -h, --help  output usage information

```

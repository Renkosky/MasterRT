# MRI-CLI

这是一套命令行工具辅助 MRI 快速开发

它包含了一切可以包含的··~~

## install

``` bash

$ npm i mri-cli -g
# version
$ mri -V

```

### 启动，测试，生产

- 启动开发环境: [mri dev :theme](/cli/mri-dev)

- 打包测试环境: [mri build :theme](/cli/mri-build)

- 打包发布环境: [mri prod :theme](/cli/mri-prod)

### 基于 git flow 流程命令

- 创建release分支 [mri git release :theme](/cli/mri-git)

- 创建hotfix分支 [mri git hotfix :theme](/cli/mri-git)

- 创建feature分支 [mri git feature :theme](/cli/mri-git)

- 自动merge [mri git merge :branch](/cli/mri-git)

### 辅助命令，快速构建文件

- 自动创建mri系统 [mri new :project](/cli/mri-new)

- 自动创建theme [mri theme :theme](/cli/mri-theme)

- 自动生成component [mri component :name](/cli/mri-component)

- 自动生成widget [mri widget :name](/cli/mri-widget)

- 自动生成interface接口 [mri interface](/cli/mri-interface)

- 自动生成index.ts文件 [mri index](/cli/mri-index)

# MrEchartsPanel

它是一组组件的集合，其最大特点是添加了一组widgets对Echarts图表进行操作

1. <Link to='/mrt/mr-echarts'> MrEcharts </Link>
2. <Link to='/mrt/mr-req'> MrReq </Link>
3. <Link to='/mrt/mr-panel'> MrPanel </Link>


## Basic usage

<Playground>
    {() => {
        let demo = MrResource.pool('/assets/pie.json')
        let $pool = {demo};
        // 将$pool存储到全局配置中供给MrReq调用
        MrServices.setResourcePool($pool);
        let req = {
            api: 'demo',
            method: 'get'
        }

        return (
            <MrEchartsPanel
                title="MrEchartsPanel::pie"
                style={{height: 300}}
                req={req} chartTypes="pie::ring">
            </MrEchartsPanel>
        );
    }}
</Playground>
